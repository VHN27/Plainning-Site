# 🗓️ Meeting Planner

[![Node.js](https://img.shields.io/badge/Backend-Node.js-339933.svg)](https://nodejs.org/)
[![PostgreSQL](https://img.shields.io/badge/Database-PostgreSQL-4169E1.svg)](https://www.postgresql.org/)

> A scalable, full-stack collaborative web application designed to streamline group scheduling and agenda management. Engineered with a robust RESTful backend and a highly optimized relational database for seamless real-time coordination.
---

## 🏗️ Architecture & Design
The application utilizes a classic client-server architecture, cleanly separating data persistence, business logic, and user interface rendering for optimal maintainability.

*   **Backend (API Layer):** Built with **Node.js** and **Express.js**, exposing RESTful endpoints to process scheduling requests, handle state, and serve client data.
*   **Database Layer:** An optimized **PostgreSQL** relational schema ensuring strict data integrity and efficient querying across complex joins (users, meetings, agendas).
*   **Frontend (Presentation):** A responsive, mobile-first interface utilizing **Bootstrap** and vanilla **JavaScript** for dynamic DOM manipulation and cross-device compatibility.

---

## 🚀 Key Technical Features

### 🔄 Scalable REST API & Real-Time Scheduling
Architected a robust backend framework to handle high-frequency, real-time meeting scheduling requests. The API ensures secure and rapid data transfer between the client and the persistent storage layer.

### 🗄️ Relational Schema Optimization
Designed a highly structured relational database in PostgreSQL. Implemented strict constraints and normalization to guarantee data integrity across interconnected entities, including user profiles, historical meeting logs, and collaborative agendas.

### 📱 Responsive & Dynamic UI
Engineered a seamless, cross-device user experience. By leveraging Bootstrap's grid system alongside asynchronous JavaScript fetch requests, the interface updates dynamically without requiring full page reloads, mirroring modern single-page application (SPA) behavior.
